# AI 활용 내역 기록

사용자의 입력을 기반으로 AI_USAGE.md 파일에 활용 내역을 추가합니다.

## 입력 형식
$ARGUMENTS

## 작업 지침

1. AI_USAGE.md 파일을 읽습니다
2. `<!-- AI_LOG_MARKER -->` 마커 바로 위에 새로운 기록을 추가합니다
3. 다음 형식으로 기록합니다:

```markdown
### {오늘 날짜}: {작업 제목}

**프롬프트:**
```
{사용자가 입력한 프롬프트 또는 요청 내용}
```

**AI 작업 내용:**
- {수행한 작업 목록}

**의사결정:**
- {왜 이런 선택을 했는지}

**수정 사항:**
- {AI 결과물을 수정한 내용과 이유, 없으면 "(없음)"}

---
```

4. 파일을 저장합니다
5. 추가된 내용을 사용자에게 확인시켜줍니다

## 예시

입력: `로그인 폼 구현 - React Hook Form 사용, 유효성 검증 추가`

결과:
```markdown
### 2026-04-03: 로그인 폼 구현

**프롬프트:**
```
로그인 폼 구현해줘
```

**AI 작업 내용:**
- React Hook Form을 사용한 로그인 폼 컴포넌트 생성
- 이메일/비밀번호 유효성 검증 추가

**의사결정:**
- React Hook Form 선택 (성능 및 DX 우수)
- Zod 스키마 기반 검증 (타입 안정성)

**수정 사항:**
- (없음)

---
```
